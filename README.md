# 🎓 KhmerKid - Nền tảng học chữ Khmer thông minh

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-Android-green.svg)](https://www.android.com/)

Nền tảng học chữ Khmer thông minh dành cho học sinh tiểu học. Học tập hiệu quả, trực quan và thú vị với công nghệ AI tiên tiến.

## ✨ Tính năng chính

- 📚 **Học phụ âm** - Làm quen với 33 phụ âm Khmer qua hình ảnh sinh động
- 🔤 **Học nguyên âm** - Phân biệt nguyên âm độc lập và nguyên âm ghép
- 🔗 **Ghép vần** - Quy tắc ghép phụ âm và nguyên âm thành từ có nghĩa
- 👂 **Luyện nghe** - Nghe và nhận diện âm tiết thông qua bài tập tương tác
- 🎤 **Luyện nói (AI)** - Công nghệ AI đánh giá phát âm ngay lập tức
- ✍️ **Luyện viết** - Học cách viết từng nét chữ theo quy chuẩn
- 🎮 **Trò chơi** - Vừa học vừa chơi với kho mini-games đa dạng
- ⭐ **Hệ thống phần thưởng** - Tích lũy Sao và XP để mở khóa cấp độ mới
- 📊 **Theo dõi tiến độ** - Báo cáo chi tiết kết quả học tập
